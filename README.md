# 🧠 Fake Job Detector Project

A web-based project that helps users identify whether a job posting is **real or fake** using simple rule-based logic and keyword analysis.

## 📌 About the Project
Fake job postings are increasing on online platforms. This project aims to help users identify suspicious job listings by analyzing patterns, keywords, and job description structure.

## ✨ Features
- Detects fake job postings using rule-based logic  
- Simple and user-friendly interface  
- Fast results without login required  
- Responsive design (works on mobile & desktop)
